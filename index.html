ETK Global Integrated Engineering Portfolio

Halaman ini mendokumentasikan pembuktian keahlian, dekonstruksi pemecahan masalah (*problem-solving*), serta arsitektur pengembangan integrasi kode yang telah diselesaikan pada proyek ini.

1. Resolusi Jalur Distribusi Aset (Raw Image CDN)

Mengatasi isu kegagalan pemuatan gambar (*broken links*) akibat proteksi intermitten pada hosting statis GitHub Pages dengan mengonversi jalur repositori mentah ke dalam format CDN *production-ready*:

  • https://github.com/.../blob/main/logo_etk_cr.png (Format salah/terblokir)
  • Direstrukturisasi total menjadi: https://raw.githubusercontent.com/gasfarrasyid/etk_bisa/main/logo_etk_cr.png

2. Implementasi CSS Zero-Margin Navigation Backdrop Layer

Memasang gambar aset dekoratif etk_future.png agar merekat presisi tanpa celah (Zero Edge) pada koordinat kiri, atas, dan bawah bar navigasi utama. Pengaturan lapisan menggunakan komposit CSS khusus:

  • Mengisolasi ruang koordinat dengan properti position: absolute; left: 0; top: 0; bottom: 0;
  • Memanfaatkan teknik modern mask-image: linear-gradient(...) untuk menghasilkan efek gradasi memudar transparan yang halus ke arah kanan, menjamin teks menu navigasi tetap terbaca jelas (*high-readability*).

3. Desentralisasi Arsitektur Sub-Domain Terisolasi

Memisahkan ruang lingkup program IELTS Preparation ke dalam gerbang navigasi fungsional terpisah (Placement Test, Review Hasil, Form Data Diri) dengan menonaktifkan seluruh navigasi makro korporasi saat sistem mendeteksi perpindahan halaman aktif.
